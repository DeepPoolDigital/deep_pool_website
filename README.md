# DEEP POOL Advertising Agency Website

A modern, responsive website for an advertising agency called DEEP POOL, inspired by the layout and design of creativemonkeys.in.

## Features

- Responsive design that works on all devices
- Modern and clean layout with grid-based sections
- Portfolio grid to showcase creative work
- Services section with detailed descriptions
- Team members showcase
- Testimonials from clients
- Client logos section
- Contact form
- Smooth scrolling navigation
- Animated elements on scroll

## Structure

- `index.html` - Main HTML file with the website structure
- `css/style.css` - CSS styles for the website
- `js/script.js` - JavaScript for interactivity and animations

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser to view the website
3. Customize the content, images, and styles to fit your needs

## Customization

### To add your own images:
1. Replace the placeholder text (e.g., "Image 1", "Image 2") with actual images
2. Update the CSS to style your images appropriately

### To change colors:
1. The primary color is set to `#ff6b6b` - you can change this in the CSS file
2. Update other color values in the CSS file to match your brand

### To update content:
1. Edit the text in `index.html` to reflect your services, team, and testimonials
2. Add your own client logos in the clients section
